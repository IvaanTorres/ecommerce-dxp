## Install for development

- Install dependencies

```bash
npm i
```

- Start the development server

```bash
npm run dev
```

- (OPTIONAL) - Start the Storybook server

```bash
npm run storybook
```

- (OPTIONAL) - Lint code

```bash
npm run lint
```

- (OPTIONAL) - Run tests

```bash
npm run tests
```
